# Build a RESTful API with keystonejs v4

Lean how to build a Nodejs RESTful APIs with KeystoneJS; a Node.js CMS (Content Management System) and web app framework built with Express and Mongoose ODM (Object-Document Mapper). The API can be used as a backend to any mobile or web application

## Directory structure
```
  .|
  --| models/
  ----| Todo.js
  ----| User.js
  --| routes/
  ----| api/
  ------| todos.js
  ----| index.js
  --| updates/
  ----| 0.0.1-first-user.js
  --| keystone.js
  --| package.json
  --| yarn.lock (or package-lock.json)
```
